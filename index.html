Flujo de la Quantum HoopSync

Cliente (Jugador)

Los jugadores acceden al juego a través del navegador, con una interfaz desarrollada en React y p5.js que ofrece una experiencia visual y funcional de alta calidad.

GitHub

El código fuente se almacena en un repositorio de GitHub, permitiendo control de versiones y despliegues automáticos a través de Render.

Render (Servidor)

Render aloja el servidor Node.js, que gestiona la lógica del juego, las conexiones WebSocket y la comunicación entre jugadores.

MongoDB Atlas

MongoDB Atlas proporciona una base de datos en la nube para almacenar rankings y estadísticas, con acceso eficiente para el servidor.
